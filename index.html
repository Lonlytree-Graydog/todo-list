📖 使用帮助

🖐️ 表单键盘操作

  • Tab：在表单字段间前进，折叠状态下的隐藏字段自动跳过。
  • Shift + Tab：在表单字段间后退。
  • 方向键：在下拉框、日期选择器中切换选项。
  • 空格：勾选/取消复选框，打开下拉框。
  • Enter：提交表单、确认下拉选项。

📋 添加任务

  • 在顶部输入框填写标题，选择优先级（🔴高/🟡中/🟢低），点击"添加任务"或按 Enter。
  • 点击"展开更多"可设置：截止日期、分类（工作/学习/生活/其他）、所属文件夹、预计时长（分钟）。
  • 截止日期支持中文格式显示（如 6月18日），过期自动标红。

✏️ 编辑与操作

  • 完成：点击任务左侧方框打勾，已完成任务显示删除线。
  • 编辑：点击 ✏️ 按钮或双击任务，修改后保存。
  • 删除：点击 🗑 按钮，或右键任务选择删除。
  • 多字段：编辑时可修改标题、描述、优先级、截止日期、分类、文件夹、时长。

🔍 筛选与搜索

  • 状态筛选：工具栏切换"全部/进行中/已完成"。
  • 搜索：按标题和描述进行模糊匹配，支持中文。
  • 紧凑已完成：点击 ✓ 按钮切换紧凑模式，已完成任务缩略显示为单行。

📋 子任务

  • 任务卡片中展开▸ 子任务，可为任务创建子步骤。
  • 每个子任务有独立标题、时长和完成状态，继承父任务的优先级和截止日期。
  • 子任务总时长不得超过父任务总时长。
  • 右键任务 → 成为子任务：将任务转为另一个任务的子任务，按时间分组显示可选父任务。

📁 文件夹管理

  • 创建：点击文件夹栏右侧的 ＋ 按钮，输入名称（最多20字）确认。
  • 切换：点击文件夹标签查看该文件夹内的任务。
  • 重命名：右键文件夹标签 → 重命名。
  • 删除：右键文件夹标签 → 删除。文件夹内任务不会丢失，自动归为"未分类"。

📌 任务归属

  • 添加或编辑任务时，可选择所属文件夹和分类标签。
  • 移动任务：右键任务 → 移动到文件夹 → 选择目标文件夹。
  • 分类标签（工作/学习/生活/其他）和文件夹是独立维度，可叠加筛选。

🔄 重复任务

  • 勾选"重复任务"，设置开始和结束日期，选择重复模式：每天/工作日/休息日/每周指定日。
  • 每天首次打开页面时，自动为当天日期创建一个独立任务实例（符合重复条件时）。
  • 若设定了总时长，每个推送实例自动按总时长 / 周期天数均分。

📊 规则管理

  • 点击文件夹栏"🔄 每日任务"标签查看所有规则，含进度条和状态。
  • ✏️ 修改：编辑规则标题、优先级、分类、日期范围、重复模式。
  • 🔄 重推：若误删今日任务实例，可手动重新推送。
  • ⠿ 拖拽：按住 ⠿ 可拖拽排序规则列表，顺序自动保存。
  • 🗑 删除规则不影响已生成的普通任务实例。

🔍 追踪任务

  • 添加任务时勾选"追踪"，必须设置截止日期。与每日重复互斥。
  • 截止日期未完成任务，次日自动删除旧实例、生成当天新任务，始终只保留一条。
  • 标记完成后追踪自动停用；删除任务时追踪规则同步移除。
  • 任务卡片显示🔍 追踪中标签，工具栏统计区显示🔍 N 追踪中。

📊 排序方式

  • 工具栏排序下拉框支持：创建时间 ↓↑、截止日期 ↓↑、优先级 ↓↑、手动排序。
  • 手动排序：在任务区域按住鼠标拖拽即可调整顺序，拖拽后自动切换到手动排序模式。
  • 排序方式会保存在浏览器中，下次打开自动恢复。

📅 日期分组

  • 点击工具栏 📅 分组 按钮开启/关闭日期分组视图。
  • 自动按截止日期归类为：已过期 / 今天 / 明天 / 本周 / 下周 / 更远 / 无截止日期。
  • 分组标题旁显示具体日期范围，支持点击折叠/展开该组。
  • 分组模式下每个组内支持拖拽排序。
  • 可与文件夹筛选、状态筛选、搜索叠加使用。

🖱️ 拖拽排序细节

  • 在任务卡片区域按住鼠标左键拖拽，松开即可完成排序。
  • 拖拽过程中会显示半透明跟随卡片和目标插入位置指示。
  • 按 Esc 可取消正在进行的拖拽操作。

💾 导入 / 导出

  • 导出：点击底部"导出"按钮，将所有任务、文件夹、每日任务规则保存为 JSON 文件。
  • 导入：点击"导入"按钮，选择之前导出的 JSON 文件，恢复所有数据。
  • 导入会覆盖当前所有数据，请谨慎操作。

🔒 数据安全

  • 所有数据保存在浏览器的 localStorage 中，关闭页面不会丢失。
  • 数据不会上传到任何服务器，仅供本地使用。
  • 清除浏览器缓存或使用隐私模式可能会丢失数据，建议定期导出备份。

🎨 主题切换

  • 左键点击标题栏的 🌙/☀️ 按钮或按 Ctrl + Alt + T 切换亮色/暗色模式。
  • 主题偏好会自动保存在本地，下次打开自动恢复。

🖼️ 背景切换

  • 右键点击标题栏的 🌙/☀️ 按钮或按 Ctrl + B 可依次切换 6 张背景图片。
  • 切换偏好会自动保存，下次打开保持上次选择的背景。
